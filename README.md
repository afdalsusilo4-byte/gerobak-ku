# 🛒 Gerobak Ku

**Gerobak Berkah Untuk Kita Semua**

Landing page profesional untuk jasa pembuatan gerobak — Food & Beverage, Kiosk Counter Mall, dan Custom Design.

## 🌐 Live Demo

🔗 [gerobak-ku.vercel.app](https://gerobak-ku.vercel.app)

## 📋 Fitur

- ✅ **Hero Section** — Tagline, CTA WhatsApp, statistik
- ✅ **Layanan** — 3 kartu layanan (F&B, Kiosk Mall, Custom)
- ✅ **Galeri Portofolio** — 6 foto gerobak dengan hover effect
- ✅ **Proses Pemesanan** — 4 langkah cara pesan
- ✅ **Keunggulan** — Kenapa pilih Gerobak Ku
- ✅ **Kontak & CTA** — WhatsApp, jam kerja, lokasi
- ✅ **Floating WhatsApp Button** — Selalu terlihat
- ✅ **Responsive** — Mobile-first design
- ✅ **Animasi** — Scroll reveal, hover effects, counter
- ✅ **Dark Theme** — Silver & Coklat

## 🛠️ Tech Stack

- HTML5
- Tailwind CSS v4 (CDN)
- Lucide Icons
- Google Fonts (Playfair Display + Inter)
- Unsplash Images

## 🚀 Deploy

### Vercel (Recommended)
1. Push ke GitHub
2. Import di [vercel.com](https://vercel.com)
3. Auto deploy!

### GitHub Pages
1. Settings → Pages
2. Source: `main` branch
3. Save

## ✏️ Customization

### Ganti Nomor WhatsApp
Cari `628xxxxxxxxx` di `index.html` dan ganti dengan nomor Anda.

### Ganti Foto
Ganti URL Unsplash di tag `<img>` dengan foto gerobak Anda sendiri.

### Ganti Harga
Edit bagian harga di section layanan (mulai dari Rp X Juta).
